# 🌐 Social Media Website

A full-stack social media web application where users can connect, share posts, like, comment, and interact with each other in real time.

---

## 🚀 Features

- 🔐 User Authentication (Register / Login)
- 👤 User Profiles
- 📝 Create, Edit & Delete Posts
- ❤️ Like & Comment on Posts
- 👥 Follow / Unfollow Users
- 📰 News Feed
- 📷 Image Upload (Posts & Profile Picture)
- 🔔 Notifications (Optional)
- 🔍 Search Users & Posts
- 📱 Responsive Design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3 / Tailwind / Bootstrap
- JavaScript (ES6+)
- Axios

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Other Tools
- JWT Authentication
- bcrypt (Password Hashing)
- Multer / Cloudinary (Image Uploads)
- Git & GitHub
